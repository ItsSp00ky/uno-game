# Custom Card Backs

Put your card-back images in this folder.

## Supported formats

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`

## Recommended size

- `400 x 620` pixels (UNO card ratio `1 : 1.55`)
- Keep all images the same ratio to avoid stretching

## Notes

- File names should be unique.
- After adding/removing images, restart `unapy` to refresh the list.
- Players choose from these files inside the in-game menu (text-only list of names).
